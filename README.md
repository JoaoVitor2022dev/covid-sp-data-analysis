# Projeto Análise dos Dados do COVID-19 no Estado de São Paulo

Este projeto realiza uma análise exploratória dos dados de COVID-19 no estado de São Paulo utilizando Python e Jupyter Notebook.

O objetivo é estudar a evolução dos casos, óbitos e indicadores epidemiológicos através de técnicas de análise de dados.

---

# Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

# Base de Dados

Os dados utilizados neste projeto foram disponibilizados pela Fundação SEADE.

Fontes:

- https://www.seade.gov.br/coronavirus/
- https://github.com/seade-R/dados-covid-sp

---

# Principais Análises

- Importação e tratamento de dados
- Limpeza de dados
- Análise exploratória
- Casos confirmados
- Número de óbitos
- Evolução temporal da pandemia
- Indicadores por município
